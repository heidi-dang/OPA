## 🎯 Overview

The **OPA (Offensive Penetration Agent)** is a comprehensive security auditing plugin for OpenCode that provides:

- **🔒 Privacy-First Operations**: Complete anonymity enforcement
- **🛡️ Security Analysis**: Vulnerability scanning and assessment
- **🔍 Intelligence Gathering**: OSINT and reconnaissance capabilities  
- **🕵️ Anonymous Agent**: Orchestrated workflow automation
- **🌐 Web Integration**: Multiple search engines and sources
- **⚡ Parallel Execution**: Concurrent task processing

### Key Features:
- ✅ **Local Sandbox Environment** with Docker isolation
- ✅ **Tor Integration** for complete IP masking
- ✅ **25+ Security Tools** for comprehensive testing
- ✅ **Intelligent Automation** with workflow orchestration
- ✅ **Real-time Reporting** with detailed analytics

## 🚀 Quick Start

### 1. Installation
```bash
# Clone the repository
git clone https://github.com/your-org/OPA.git
cd OPA

# Install dependencies
npm install

# Build the plugin
npm run build

# The plugin is now ready for use in OpenCode!
```

### 2. Basic Usage in OpenCode

Once installed, the OPA plugin will appear in your OpenCode tools panel. You can:

1. **Access Tools Panel**: Look for "OPA" in your OpenCode extensions
2. **Select Tool**: Choose from 25+ available security tools
3. **Configure Parameters**: Set targets and options as needed
4. **Execute**: Run tools with approval workflow
5. **Review Results**: Get comprehensive reports and recommendations

## 🐛 Troubleshooting

### **Common Issues & Solutions**

#### **Sandbox Issues**
```bash
# Problem: Sandbox not deploying
# Solution: Check Docker status
docker --version
sudo systemctl status docker

# Problem: Tools not working in sandbox
# Solution: Restart sandbox
stop_privacy_services
deploy_sandbox_as_needed
```

#### **Privacy Issues**
```bash
# Problem: Tor not connecting
# Solution: Check Tor service
systemctl status tor
sudo systemctl restart tor

# Problem: IP not masked
# Solution: Verify configuration
check_privacy_status
quick_privacy_check
```

#### **Build Issues**
```bash
# Problem: TypeScript compilation errors
# Solution: Clean build
npm run clean
npm run build

# Problem: Missing dependencies
# Solution: Install dependencies
npm install
```

```bash

check_privacy_status

reset_agent_deployment
```
